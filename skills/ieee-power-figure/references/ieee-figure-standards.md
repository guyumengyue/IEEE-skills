# IEEE Figure Standards

Use these rules for IEEE PES Transactions figures.

## Size

- One-column graphics: 3.5 in / 88.9 mm / 21 picas.
- Two-column graphics: 7.16 in / 182 mm / 43 picas.
- Avoid creating graphics below one-column width because IEEE can reduce
  graphics more safely than enlarge them.

## Resolution and formats

- Prefer vector graphics for charts and diagrams: PDF, EPS, or PS.
- IEEE accepts PS, EPS, PDF, PNG, and TIFF for graphics.
- For non-vector color and grayscale images, use more than 300 dpi.
- For black-and-white line art, use more than 600 dpi.
- Embed fonts and keep text editable where possible.

## Accessibility

- A line graph must remain interpretable in grayscale.
- Use both color and shape: line style, marker, fill pattern, or direct label.
- Use unique data symbols for each line.
- Use sufficient contrast in both hue and brightness.
- Directly label curves when doing so reduces legend eye travel.

## IEEE/PES notes

- PES color figures are processed online in color free of charge.
- Do not use the Lena image in submitted papers.
- Keep captions factual: quantities, units, scenarios, error bars, and sample
  sizes belong in the caption or surrounding text.
