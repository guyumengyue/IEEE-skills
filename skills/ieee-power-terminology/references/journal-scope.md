# Journal Scope Wording

## TPWRS wording

Use phrases such as:

- power system operation
- dynamic security assessment
- power system planning
- transmission and distribution system analysis
- reliability and resilience
- uncertainty-aware optimization
- state estimation and control

## TSG wording

Use phrases such as:

- active distribution network operation
- microgrid energy management
- DER and EV grid integration
- AMI-enabled monitoring
- distribution PMU application
- cyber-physical smart-grid security
- transactive energy coordination
- data analytics for microgrids and ADNs

## Scope-risk flags

Flag these for TSG unless clearly connected to smart-grid scope:

- transmission-only load or price forecasting
- WAMS/WACS transmission applications
- transmission protection
- DC transmission systems
- market framework issues not tied to DR/DSM, ADNs, microgrids, DERs, EVs, or
  multi-energy systems
