# IEEE PES Term Map

## Preferred terms

| Prefer | Avoid or check |
|---|---|
| power flow analysis | load flow calculation |
| optimal power flow | optimal powerflow |
| microgrid | micro-grid |
| distributed energy resources | distributed generations when DERs include storage or flexible load |
| EV chargers | charging piles |
| battery energy storage system | storage battery system |
| active distribution network | active distribution grid unless the paper defines grid differently |
| demand-side management | demand side management when used as a compound adjective |
| phasor measurement unit | phase measurement unit |
| voltage regulation | voltage control when the distinction matters |

## Acronyms

Define on first use:

- ADN: active distribution network
- AMI: advanced metering infrastructure
- BESS: battery energy storage system
- DER: distributed energy resource
- DR: demand response
- DSM: demand-side management
- EV: electric vehicle
- OPF: optimal power flow
- PMU: phasor measurement unit
- WAMS: wide-area measurement system
- WACS: wide-area control system

## Units and notation

- Use p.u. with the base quantity defined.
- Use MW, Mvar, kV, Hz, and s consistently.
- Avoid mixing bus, node, feeder, and area unless each has a defined meaning.
