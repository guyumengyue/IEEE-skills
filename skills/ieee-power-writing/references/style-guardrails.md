# IEEE PES Style Guardrails

- Use SI units and define per-unit bases.
- Define acronyms on first use unless universally standard in the target venue.
- Use "power flow analysis" rather than "load flow calculation" in modern prose.
- Use "microgrid" rather than "micro-grid".
- Avoid unsupported "first", "optimal", "real-time", or "robust" claims.
- Prefer "reduces operating cost by X%" over "significantly improves economy"
  when the data are provided.
- Distinguish forecast, estimation, control, optimization, and protection.
- Keep limitations concrete: system size, operating condition, uncertainty model,
  communication assumptions, data availability, or hardware validation boundary.
